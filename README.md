
# 🛍️ Customer Shopping Data Analysis

##  Project Overview
This project focuses on analyzing customer shopping behavior across multiple shopping malls between 2021 and 2023. The goal is to extract meaningful insights related to customer demographics, purchasing patterns, and revenue generation.

---

##  Objectives
- Analyze shopping distribution based on gender and age
- Identify which gender generates more revenue
- Analyze product category trends
- Study payment method distribution
- Provide business insights and recommendations

---

## Technologies Used
- **Python (Jupyter Notebook)**
- **MySQL**
- **Power BI**
- Libraries: Pandas, NumPy, Matplotlib

---

##  Data Analysis Process
1. Data Collection from MySQL database  
2. Data Cleaning and Preprocessing using Python  
3. Exploratory Data Analysis (EDA)  
4. Data Visualization using Power BI  
5. Insight Generation  

---

##  Key Insights
- Identified customer segments contributing most to revenue  
- Analyzed age-wise and gender-wise purchasing behavior  
- Observed popular product categories  
- Evaluated payment method usage trends  

---


##  Business Recommendations
- Focus marketing strategies on high-revenue customer segments  
- Optimize product categories based on demand trends  
- Improve payment options for better customer experience
